## ✨ Key Features (주요 기능)

- **고밀도 프로젝트 리스트**: 35개 이상의 방대한 프로젝트 포트폴리오를 한눈에 탐색할 수 있는 마이크로 썸네일 기반의 리스트 레이아웃을 구현했습니다.
- **프리미엄 인터랙션**: GSAP 애니메이션 라이브러리를 활용하여 매끄러운 스크롤 효과와 고대비 브랜드 블루 호버 액션을 통해 고급스러운 사용자 경험을 제공합니다.
- **반응형 상세 갤러리**: 모든 프로젝트 상세 페이지에 반응형 그리드 시스템을 적용하여, PC와 모바일 환경 모두에서 최적화된 이미지 뷰를 제공합니다.
- **완벽한 반응형 설계**: 데스크탑, 태블릿, 모바일 등 모든 기기 환경에서 일관된 디자인과 기능을 유지할 수 있도록 세밀하게 설계되었습니다.
- **성능 및 코드 최적화**: Next.js의 이미지 최적화 및 최신 React 기능을 활용하여 빠른 로딩 속도와 유지보수가 용이한 컴포넌트 구조를 지향합니다.

## 🛠 Tech Stack (기술 스택)

- **Framework**: `Next.js 16 (App Router)`, `React 19`
- **Styling**: `Tailwind CSS 4`, `Vanilla CSS`
- **Animation**: `GSAP`, `@gsap/react`
- **Icons**: `Lucide React`
- **Language**: `TypeScript`

## 📂 Project Structure (폴더 구조)

```bash
src/
├── app/          # Next.js App Router (페이지 및 라우팅)
├── components/   # 재사용 가능한 UI 컴포넌트
│   ├── layout/   # Header, Footer 등 레이아웃 구성 요소
│   ├── sections/ # Hero, About, Portfolio, Contact 등 섹션별 컴포넌트
│   └── project/  # 프로젝트 상세 전용 컴포넌트
├── styles/       # 글로벌 CSS 및 테마 설정
└── utils/        # 프로젝트 데이터 (projects.ts) 및 공통 함수
public/
└── images/       # 프로젝트 에셋 및 아이콘
```
