# 임현욱 | Web Publisher Portfolio

41+ 프로젝트 경험을 보유한 4년차 웹퍼블리셔 임현욱의 포트폴리오 사이트입니다. 기획의 본질을 파악하고 단단한 코드로 안정적인 웹 환경을 구축하는 것을 목표로 합니다.

## 🚀 Teck Stack

- **Core**: Next.js 14, TypeScript
- **Styling**: Tailwind CSS, Vanilla CSS
- **Animation**: GSAP (ScrollTrigger), Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel

## ✨ Key Features

### 1. Unified Line Animation
- Hero 섹션부터 사이트 끝까지 이어지는 단일 곡선 애니메이션을 통해 프로젝트의 핵심 키워드인 '연결(잇다)'을 시각화했습니다.

### 2. Identity (About) Section
- **Unified Experience**: 'Core Skills'가 'About' 섹션으로 통합되어 교육, 경력, 기술 스택을 한눈에 파악할 수 있는 몰입형 레이아웃을 제공합니다.
- **GSAP ScrollTrigger**: 스크롤에 반응하는 텍스트 리빌(Reveal) 효과와 프로필 이미지 패럴랙스 효과를 구현했습니다.
- **Interactive Profile**: 마우스 호버 시 일러스트에서 원본 사진으로 부드럽게 전환되는 크로스페이드 효과를 적용했습니다.
- **Wider Wireframe Design**: `max-w-[1440px]`의 넓은 컨텐츠 영역을 통해 시원하고 전문적인 와이어프레임 감성을 전달합니다.

### 3. Tech Stack & Portfolio
- **14+ Tech Icons**: Tailwind, SCSS, GnuBoard, PHP, MySQL 등 14개 이상의 기술 아이콘을 커스텀 제작하여 일관된 톤앤매너로 배치했습니다.
- **Project Grid**: 41건 이상의 프로젝트 데이터 중 핵심 성과를 선별하여 인터랙티브한 그리드 형태로 노출합니다.

### 4. Dedicated Contact Section
- 사용자 접근성을 높이기 위해 하단에 독립된 연락처 섹션을 배치하고 SNS 및 이메일 연결성을 강화했습니다.

## 🛠 Installation & Setup

```bash
# 의존성 설치
npm install

# 로컬 개발 서버 실행
npm run dev
```
