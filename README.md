# IM HYUNWOOK - Portfolio 🚀

안녕하세요! 웹 퍼블리셔 및 프론트엔드 개발자 **임현욱**의 포트폴리오 저장소입니다.  
본 프로젝트는 모던 웹 기술을 사용하여 제작되었으며, 35개 이상의 프로젝트 실무 경험을 효과적으로 전달하기 위해 설계되었습니다.

---

## ✨ Key Features (주요 기능)

- **High-Density Project List**: 35개 이상의 프로젝트를 효율적으로 탐색할 수 있는 마이크로 썸네일 리스트 구현.
- **Premium Interaction**: GSAP를 활용한 매끄러운 스크롤 애니메이션 및 고대비 브랜드 블루 호버 효과.
- **Project Detail Gallery**: 반응형 그리드 시스템을 적용한 상세 레이아웃 (PC & Mobile 뷰 지원).
- **Responsive Design**: 데스크탑부터 모바일 기기까지 완벽하게 대응하는 최적화된 UI/UX.
- **Performance Optimized**: Next.js의 이미지 최적화 및 최신 React 기능을 활용한 빠른 로딩 속도.

## 🛠 Tech Stack (기술 스택)

- **Framework**: `Next.js 16 (App Router)`, `React 19`
- **Styling**: `Tailwind CSS 4`, `Vanilla CSS`
- **Animation**: `GSAP`, `@gsap/react`
- **Icons**: `Lucide React`
- **Language**: `TypeScript`

## 📂 Project Structure (폴더 구조)

```bash
src/
├── app/          # Next.js App Router (페이지 및 라우팅)
├── components/   # 재사용 가능한 UI 컴포넌트
│   ├── layout/   # Header, Footer 등 레이아웃 구성 요소
│   ├── sections/ # Hero, About, Portfolio, Contact 등 섹션별 컴포넌트
│   └── project/  # 프로젝트 상세 전용 컴포넌트
├── styles/       # 글로벌 CSS 및 테마 설정
└── utils/        # 프로젝트 데이터 (projects.ts) 및 공통 함수
public/
└── images/       # 프로젝트 에셋 및 아이콘
```

## 🚀 Getting Started (시작하기)

1. **의존성 설치**:
   ```bash
   npm install
   ```

2. **개발 서버 실행**:
   ```bash
   npm run dev
   ```

3. **프로덕션 빌드**:
   ```bash
   npm run build
   ```
